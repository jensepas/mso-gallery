A lightweight and modern image gallery plugin. Includes Gutenberg block, shortcode, and elegant overlay viewer.

== Description ==

**MSO Gallery** is a simple yet powerful image gallery plugin for WordPress.
Designed for flexibility, it lets you quickly add galleries using a Gutenberg block or a classic shortcode.

Features:

- Add beautiful image galleries in posts and pages
- Includes a Gutenberg block for easy visual editing
- Classic shortcode support: `[mso_gallery ids="1,2,3"]`
- Responsive overlay/lightbox viewer in JavaScript
- Clean admin interface for managing galleries

Perfect for photographers, bloggers, or anyone who wants a fast and user-friendly gallery.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install via the WordPress plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the "MSO Gallery" menu in the admin to create galleries.
4. Insert them using the Gutenberg block or shortcode.

== Frequently Asked Questions ==

= Can I use this with the Block Editor? =
Yes! The plugin includes a native Gutenberg block for inserting galleries.

= How do I use the shortcode? =
Use `[mso_gallery ids="1,2,3"]`, replacing the IDs with your image attachment IDs.

= Does it support a lightbox/overlay viewer? =
Yes, MSO Gallery includes a built-in JavaScript overlay for viewing images.
